# Login Page UI Assignment

## Overview
This project is a simple, login page built using HTML and CSS. It was created based on an assignment focusing on basic CSS properties and modern layout techniques, specifically Flexbox.

## Technologies Used
HTML5: For structuring the document and form elements.
CSS3:: For styling, colors, and layout (Flexbox).

## Assignment Requirements Fulfilled
1. Container Setup: HTML file created, content centered on the page, container height set to `100vh`.
2. Left Section: Image added on the left side center.
3. Form Section: Form section set up inside the container using Flexbox for a neat layout.
4. Form Heading and Description: Friendly greeting ("Hello Again!") included.
5. Email and Password Inputs: Input fields for email and password added with specified types, placeholders, and clean organization.
6. Login Button: Button labelled "Login" added and styled to look appealing.
7. Constraint Check: Successfully completed the objective: *Don't use margin padding to make it in the centre. Use FlexBox.

## How to Run
1. Ensure both `index.html` and `style.css` are saved in the same directory.
2. Double-click `index.html` to open it in your default web browser, or drag and drop the file into an open browser window.

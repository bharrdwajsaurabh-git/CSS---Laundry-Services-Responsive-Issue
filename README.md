# CSS---Laundry-Services-Responsive-Issue assinment
# Overview
This project is a simple demonstration of how to make wesbite responsive using media queries.

# Files Included
**index.html: The main HTML structure containing a navbar with logo nav-links and username and also two div one with image and other with button and some information. 
**style.css: The stylesheet that applies specific rules to the HTML so it can behave responsive means the structure looks good on different screen size.

# How to Use
Ensure both index.html and style.css are saved in the same directory. Open index.html in any modern web browser to see the CSS styles applied to the HTML structure in action.

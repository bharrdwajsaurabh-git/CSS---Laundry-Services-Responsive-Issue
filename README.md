# LAUNDRYMART LOGIN PAGE USING FLEX

# Overview
This project is a simple demonstration of how CSS Display flex properties work and how we centered and change the postions the element with the help of flex properties.

# Files Included
**index.html: The main HTML structure demonstrate a login page with a main-container and two section left and right left section contains a image of washing machine and right section contains a login form and button.
**style.css: The stylesheet that applies specific rules to the HTML using Display flex properties.

# How to Use
Ensure both index.html and style.css are saved in the same directory. Open index.html in any modern web browser to see the CSS styles applied to the HTML structure in action.

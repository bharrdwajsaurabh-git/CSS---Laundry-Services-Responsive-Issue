# Laundry Service - CSS Hero Section Assignment

# Overview
This project is a simple demonstration of how CSS Display, position and float properties works.

# Files Included
**index.html: The main HTML structure containing a navbar with logo nav-links and username and also two div one with image and other with button and some information.
**style.css: The stylesheet that applies specific rules to the HTML using Display position and float properties.
# How to Use
Ensure both index.html and style.css are saved in the same directory.
Open index.html in any modern web browser to see the CSS styles applied to the HTML structure in action.
